Projects | Brilliance



    About the project
    Brilliance is a 4WD(4 Wheel Drive) robot development platform with an ATmega 644P microcontroller as it's brain onboard a custom made AVR development board.

    The development board alone can be used for developing embedded applications involving high-speed wireless communication(SPI, I2C and UART), human-computer interaction, working with sensors and motor control, etc.

    What's so special?
    The development board is fully compatible with the open source prototyping platform Arduino. You can use most of the popular Atmel's microcontrollers* with Arduino bootloader burned on it.

    Its compatible with Arduino Hardware too. Additionally, not only, you can stack up any Arduino sheild on it but also use this as a shield on top of an existing Arduino stack, by soldering male/female header pins on appropriate side.

    Features & Specifications
    2 x DRV8833 Motor Drivers for driving two brushed DC/Servo motors. Each DRV8833 Motor driver is in parellel configuration to provide maximum output current per motor (2.4 A). Active Buzzer connected to Digital pin 03. Connection for OLED Bluetooth communication, NRF Modules, OLED Display module shield to it, as It features two DRV8833 Motor Drivers with parellel configurations each to give maximum power to the motors.
